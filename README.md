# Blogging_Repo

Personal blogging repository with Markdown source and hosted HTML pages.

## Structure

- `index.html` - Home page for GitHub Pages
- `LLM_Series/1st-page.md` - Markdown source content
- `LLM_Series/1st-page.html` - Published HTML page
- `LLM_Series/images/generated/` - SVG diagrams used in posts

## Publish Workflow

1. Write/update Markdown in `LLM_Series/*.md`.
2. Convert or update corresponding HTML page.
3. Commit and push to `main`.
4. Serve via GitHub Pages.

## GitHub Pages

Set repository Pages source to:

- Branch: `main`
- Folder: `/ (root)`

Then your site will be available at:

- `https://anshul22122001.github.io/Blogging_Repo/`
